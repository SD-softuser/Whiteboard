Online whiteboard

Real-time whiteboard made with Next.JS and Socket.IO
## Features

- Drawing lines, circles and rectangles
- Eraser
- Undo/Redo
- Real-time mouse tracking
- Chatting
- Placing images
- Moving selected area
- Saving canvas
- Changing backgrounds
- Sharing
## Made using
- Next.JS
- Recoil
- TailwindCSS
- Socket.IO
